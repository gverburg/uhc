Download EHC

  • The current version of all compiler versions can be downloaded as [tgz] file.
    • New versions appear whenever a stable point in the development has been reached.
    • Requires GHC6.4 and the UUST library
  • Documentation will be available during fall 2005 (in the form of a forthcoming phd thesis). Currently some papers (submitted) are included in the doc subdirectory, about the following topics:
    • Ruler: Programming with Type Rules
    • Making Implicit Parameters Explicit
  • Installation:
    1. Unpack the EHC distribution, creating a directory <dist-dir> of the form <dist-date>-ehc-<dist-name>
    2. cd <dist-dir>
    3. Type 'make' for an explanation of what can be made

Download & install Ruler

  • Ruler is part of the EHC distribution
  • Installation:
    1. Unpack the EHC distribution (see above)
    2. cd <dist-dir>/ruler2
    3. Follow the instructions in the README file in that directory (or type 'make' for explanation)
